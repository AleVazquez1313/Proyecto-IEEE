# Proyecto IEEE
DESARROLLO DE UNA PLATAFORMA WEB DE APRENDIZAJE EN LÍNEA PARA EL Instituto Electoral del Estado de Querétaro, ORIENTADA A LA DIFUSIÓN DE CURSOS EN MATERIA ELECTORAL. EL SISTEMA PERMITIRÁ A LOS USUARIOS REGISTRARSE, ACCEDER A CURSOS, REALIZAR EVALUACIONES PARCIALES Y FINALES, ASÍ COMO OBTENER CONSTANCIAS DIGITALES AL CONCLUIR SATISFACTORIAMENTE LOS CURSOS. LA PLATAFORMA ESTARÁ DIRIGIDA A LA CIUDADANÍA EN GENERAL, ASPIRANTES A INTEGRARSE AL INSTITUTO Y PERSONAL ACTIVO.
